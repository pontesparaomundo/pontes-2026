# Pontes 2026 Dashboard

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/pontesparaomundo/pontes-2026.git
   cd pontes-2026
   ```
2. Install the required dependencies:
   ```bash
   # Example: For a Node.js project, run:
   npm install
   ```

## Usage Instructions
1. Start the dashboard:
   ```bash
   # Example: For a Node.js project, run:
   npm start
   ```
2. Open your browser and go to `http://localhost:3000` to view the dashboard.

## Accessing on GitHub Pages
1. Ensure the `gh-pages` branch is set up in the repository.
2. Go to the repository settings and enable GitHub Pages from the `gh-pages` branch.
3. Your dashboard will be available at `https://pontesparaomundo.github.io/pontes-2026`.
